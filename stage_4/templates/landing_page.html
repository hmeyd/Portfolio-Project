Articles utiles

Le numéro SIRET identifie chaque établissement. Le SIREN est commun à tous les établissements d’une entreprise. Le code APE reflète l’activité principale.

Le BODACC publie les annonces légales d’une entreprise : créations, ventes, liquidations judiciaires, etc.

Les statuts définissent l’activité, la répartition du capital, les pouvoirs des dirigeants et les règles internes de l’entreprise.

Définir l’activité, choisir une forme juridique, rédiger les statuts, déposer le capital, enregistrer au RCS et publier un avis de constitution.

À propos du projet

Ce projet a été réalisé dans le cadre du Portfolio Project de Holberton School. L'objectif est de faciliter l'accès aux données officielles des entreprises françaises grâce à une interface claire et moderne.
